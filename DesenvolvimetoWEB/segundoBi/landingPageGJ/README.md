# Landing Page - Projeto Acadêmico (React + TypeScript)

Este projeto é uma **landing page desenvolvida como trabalho da faculdade**, com o objetivo de colocar em prática todo o processo de estruturação de uma aplicação frontend moderna utilizando **React com TypeScript (TSX)**.

## 📚 Sobre o projeto

Durante o desenvolvimento, foram aplicados conceitos importantes de:

- Estruturação de projetos com React
- Utilização de TypeScript (TSX) para tipagem e organização do código
- Componentização de interface
- Estilização e construção de uma landing page responsiva
- Boas práticas de desenvolvimento frontend

O foco principal foi simular um ambiente real de desenvolvimento, desde a criação da interface até a publicação do projeto.

## 🚀 Deploy e hospedagem

O projeto foi hospedado na plataforma **Netlify**, que permite deploy contínuo e rápido de aplicações web.

Além disso, foi implementado um sistema de **envio de formulário (email)** utilizando os recursos nativos do Netlify Forms, permitindo que os dados enviados pela landing page sejam recebidos diretamente como notificações por email.

## 📩 Funcionalidade de contato

A landing page possui um formulário de contato integrado ao Netlify, permitindo:

- Envio de mensagens diretamente pela página
- Recebimento das informações por email
- Gerenciamento das submissões pelo painel do Netlify

## 🎯 Objetivo acadêmico

Este projeto foi desenvolvido como parte de uma atividade da faculdade, com o objetivo de:

- Praticar a estruturação de aplicações React com TypeScript
- Entender o fluxo de deploy de aplicações frontend
- Implementar funcionalidades reais como formulário de contato
- Utilizar uma plataforma de hospedagem moderna (Netlify)
